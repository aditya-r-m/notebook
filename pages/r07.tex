hl|tc| n-step Bootstrapping
The idea of bootstrapping in Temporal Difference learning can be extended beyond the single-step process using the concept of n-step return targets : \( G_{t:t+n} = \sum_{i=t+1}^{t+n} \gamma^{i - t - 1} R_i + \gamma^n V(S_{t+n}) \),
\( V_{t+n}(s) = \begin{cases} V_{t+n-1}(s) + \alpha [G_{t:t+n} - V_{t+n-1}(s)], & \textrm{if} \ \ S_t = s \\ V_{t+n-1}(s), & \textrm{otherwise} \end{cases} \)
Similar to the single step errors, we can show that MC error can also be approximated as a sum of n-step TD errors : \( \delta_{t:t+n} = \sum_{i=t+1}^{t+n} \gamma^{i-t-1}R_i + \gamma^n V(S_{t+n}) - V(S_t) \),
\( \begin{aligned} G_{t \ : \ t+n} - V(S_t) &= \sum_{i=t+1}^{t+n} \gamma^{i-t-1} R_{i} + \gamma^n G_{t+n \ : \ t+2n} - V(S_t) \\ &= \delta_{t \ : \ t+n} + \gamma^n(G_{t+n \ : \ t+2n} - V(S_{t+n})) \\ &= \delta_{t \ : \ t+n} + \gamma^n \left(\sum_{i=t+n+1}^{t+2n} \gamma^{i - t -n-1}R_i + \gamma^n G_{t+2n \ : \ t+3n} - V(S_{t+n})\right) \\ &= \delta_{t \ : \ t+n} + \gamma^n(\delta_{t+n \ : \ t+2n} + \gamma^n(G_{t+2n \ : \ t+3n} - V(S_{t+2n}))) = \ldots = \sum_{k=0}^{(T-1) / n} \gamma^{kn}\delta_{t+kn \ : \ t+(k+1)n} \end{aligned} \)
An important property of n-step returns is that their expectations is guaranteed to be a better estimate of \( v_\pi \) than \(V_{t+n-1} \) in a worst case sense,
\( \max_s | E_\pi(G_{t:t+n} | S_t = s) - v_\pi(s) | \le \gamma^n \max_s | V_{t+n-1}(s) - v_\pi(s) | \)
This property ensures convergence to correct predictions under appropriate conditions.
hs| n-step Sarsa
An \( \epsilon \)-Greedy policy \( \pi \) can be learnt using the Q-function n-step estimate as follows,
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha [ \sum_{i=0}^{n-1} \gamma^i R_{t+i+1} + \gamma^n Q_{t+n-1}(S_{t+n}, A_{t+n}) - Q_{t+n-1}(S_t, A_t) ] \)
n-step expected Sarsa can also be implemented using similar updates,
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha [ \sum_{i=0}^{n-1} \gamma^i R_{t+i+1} + \gamma^n \sum_a \pi(a|s) \ Q_{t+n-1} (S_{t+n}, a) - Q_{t+n-1}(S_t, A_t) ] \)
Note that the n-step return of Sarsa can be written in terms of a novel TD-error : \( G_{t:t+n} = Q_{t-1}(S_t, A_t) + \sum_{k=t}^{t+n-1} \gamma^{k-t}[R_{k+1} + \gamma Q_k(S_{k+1}, A_{k+1}) - Q_{k-1}(S_k, A_k)] \)
hs| n-step Off-policy Learning
Importance sampling ratio \( \rho_{t+1:t+n} = \Pi_{k=t+1}^{t+n} \frac{\pi(A_k|S_k)}{b(A_k|S_k)} \) can be used for learning the Q-function with a behavior policy \( b \) different from target policy \( \pi \),
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha \rho_{t+1:t+n} [G_{t:t+n} - Q_{t+n-1}(S_t, A_t)] \)
Note that we are excluding the factor \( \frac{\pi(A_t|S_t)}{b(A_t|S_t)} \) because \( Q_{t+n}(S_t, A_t) \) by default assumes that learning is specific to the case \( (A_t|S_t) \). A similar factor can also be used with Off-policy Learning for n-step expected Sarsa by excluding \( \frac{\pi(A_{t+n}|S_{t+n})}{b(A_{t+n}|S_{t+n})} \) to account for the fact that the last step of expected Sarsa is not part of the sampling process, and is a direct expectation computation instead.
Importance sampling ratio can also be embedded into the return itself using per-decision methods, eliminating the need to use it for scaling TD-error.
For basic state-value function, the simplest approach will be to define \( G_{t:h} = \rho_t (R_{t+1} + G_{t+1:h}) \) where \( \rho = \frac{\pi(A_t | S_t)}{b(A_t|S_t)} \). An improvement on this that can reduce variance significantly is to add a term called control variate as follows : \( G_{t:h} = \rho_t(R_{t+1} + G_{t+1:h}) + (1 - \rho_t) V_{h-1}(S_t) \)
For action-value function, the current Reward will not be scaled, and the control variate will need to account for the fact that recursion has to end with the expected value of returns,
\( G_{t:h} = R_{t+1} + \gamma( \rho_{t+1} G_{t+1:h} + \bar{V}_{h-1}(S_{t+1}) - \rho_{t+1} Q_{h-1}(S_{t+1}, A_{t+1}) ) = R_{t+1} + \gamma \rho_{t+1} (G_{t+1:h} - Q_{h-1}(S_{t+1}, A_{t+1})) + \gamma \bar{V}_{h-1}(S_{t+1}), \ \ t < h \le T \)
There is also a very natural approach called n-step Tree-backup learning, which implements Off-policy learning without importance sampling like expected Sarsa and Q-learning for single-step cases,
\( G_{t:t+n} = R_{t+1} + \gamma \sum_{a\neq A_{t+1}} \pi(a|S_{t+1}) Q_{t+n-1}(S_{t+1}, a) + \gamma \pi(A_{t+1}|S_{t+1}) G_{t+1:t+n}\)
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha[G_{t:t+n} - Q_{t+n-1}(S_t, A_t)] \)
To summarize the 3 key approaches covered for off-policy learning,
li| n-step Sarsa uses importance sampling for every step
li| n-step Expected Sarsa uses importance sampling for every step excluding the last, where it uses expected value
li| n-step Tree backup uses expected value for all the steps
These ideas can be unified starting with the observation that the return for Tree backup can be written in terms of the expected approximate value \( \bar{V} \),
\( G_{t:h} = R_{t+1} + \gamma \pi(A_{t+1} | S_{t+1}) (G_{t+1:h} - Q_{h-1}(S_{t+1}, A_{t+1}) )+ \gamma \bar{V}_{h-1}(S_{t+1}) \)
This form looks identical to n-step return for Sarsa with control variates. Combining these two using a Vector \( \sigma \), we get a unified algorithm using the following return,
\( Q(\sigma) : G_{t:h} = R_{t+1} + \gamma (\sigma_{t+1} \rho_{t+1} + (1-\sigma_{t+1}) \ \pi(A_{t+1} | S_{t+1})) (G_{t+1:h} - Q_{h-1}(S_{t+1}, A_{t+1})) + \gamma \bar{V}_{h-1}(S_{t+1}) = R_{t+1} + \gamma \bar{V}_{h-1}(S_{t+1}) + \gamma m_{t+1}(G_{t+1:h} - Q_{h-1}(S_{t+1}, A_{t+1})) \)
Note that this unified return can also be seen as a composition of single-step TD-errors \( \delta_{t} = R_{t+1} + \gamma \bar{V}(S_{t+1}) - Q(S_t, A_t) \),
\( \begin{aligned} G_{t:h} - Q(S_t, A_t) &= \delta_{t} + \gamma m_{t+1} (G_{t+1:h} - Q(S_{t+1}, A_{t+1})) \\ &= \delta_t + \gamma m_{t+1} (\delta_{t+1} + \gamma m_{t+2} (G_{t+2:h} - Q(S_{t+2}, A_{t+2}))) \\ &= \ldots = \sum_{k=0}^{h-t} \delta_{t+k} \prod_{i=k+1}^{h-t} \gamma m_{t+i} \end{aligned} \)
