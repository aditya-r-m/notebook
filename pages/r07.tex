hl|tc| n-step Bootstrapping
The idea of bootstrapping in Temporal Difference learning can be extended beyond the single-step process using the concept of n-step return targets : \( G_{t:t+n} = \sum_{i=t+1}^{t+n} \gamma^{i - t - 1} R_i + \gamma^n V(S_{t+n}) \),
\( V_{t+n}(s) = \begin{cases} V_{t+n-1}(s) + \alpha [G_{t:t+n} - V_{t+n-1}(s)], & \textrm{if} \ \ S_t = s \\ V_{t+n-1}(s), & \textrm{otherwise} \end{cases} \)
Similar to the single step errors, we can show that MC error can also be approximated as a sum of n-step TD errors : \( \delta_{t:t+n} = \sum_{i=t+1}^{t+n} \gamma^{i-t-1}R_i + \gamma^n V(S_{t+n}) - V(S_t) \),
\( \begin{aligned} G_{t \ : \ t+n} - V(S_t) &= \sum_{i=t+1}^{t+n} \gamma^{i-t-1} R_{i} + \gamma^n G_{t+n \ : \ t+2n} - V(S_t) \\ &= \delta_{t \ : \ t+n} + \gamma^n(G_{t+n \ : \ t+2n} - V(S_{t+n})) \\ &= \delta_{t \ : \ t+n} + \gamma^n(\sum_{i=t+n+1}^{t+2n} \gamma^{i - t -n-1}R_i + \gamma^n G_{t+2n \ : \ t+3n} - V(S_{t+n})) \\ &= \delta_{t \ : \ t+n} + \gamma^n(\delta_{t+n \ : \ t+2n} + \gamma^n(G_{t+2n \ : \ t+3n} - V(S_{t+2n}))) = \ldots = \sum_{k=0}^{(T-1) / n} \gamma^{kn}\delta_{t+kn \ : \ t+(k+1)n} \end{aligned} \)
An important property of n-step returns is that their expectations is guaranteed to be a better estimate of \( v_\pi \) than \(V_{t+n-1} \) in a worst case sense,
\( \max_s | E_\pi(G_{t:t+n} | S_t = s) - v_\pi(s) | \le \gamma^n \max_s | V_{t+n-1}(s) - v_\pi(s) | \)
This property ensures convergence to correct predictions under appropriate conditions.
hs| n-step Sarsa
An \( \epsilon \)-Greedy policy \( \pi \) can be learnt using the Q-function n-step estimate as follows,
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha [ \sum_{i=0}^{n-1} \gamma^i R_{t+i+1} + \gamma^n Q_{t+n-1}(S_{t+n}, A_{t+n}) - Q_{t+n-1}(S_t, A_t) ] \)
n-step expected Sarsa can also be implemented using similar updates,
\( Q_{t+n}(S_t, A_t) = Q_{t+n-1}(S_t, A_t) + \alpha [ \sum_{i=0}^{n-1} \gamma^i R_{t+i+1} + \gamma^n \sum_a \pi(a|s) \ Q_{t+n-1} (S_{t+n}, a) - Q_{t+n-1}(S_t, A_t) ] \)
Note that the n-step return of Sarsa can be written in terms of a novel TD-error : \( G_{t:t+n} = Q_{t-1}(S_t, A_t) + \sum_{k=t}^{t+n-1} \gamma^{k-t}[R_{k+1} + \gamma Q_k(S_{k+1}, A_{k+1}) - Q_{k-1}(S_k, A_k)] \)
